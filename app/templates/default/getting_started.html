Getting Started

Step 1

Download the code: 
$ git clone https://bitbucket.org/battlemidget/cage.git

Step 2

Install a skeleton project: 
$ ./path/to/cage/claw new "project name"

Step 3

Run the skeleton project to see a demo: 
$ ./projectname/claw project --runserver --foreground

Step 4

Visit site: 
http://localhost:10001
